# Dbable

[English](README.md) | 中文说明

## 关于

Dbable是一款现代化的跨平台数据库管理软件，专为开发人员和数据库管理员设计。现已支持`Windows`、`Mac`和`Linux`平台。

## 功能特性

**1. 对象设计器**
高效管理常用的数据库对象。

**2. 数据编辑器**
插入、编辑、删除、复制或粘贴数据记录。支持编辑查询结果。

**3. 代码补全**
智能补全关键字、函数、数据库、表、字段和别名，提升编码速度。

**4. 代码片段**
管理常用代码片段，提高工作效率。

**5. 导入/导出**
支持导入和导出csv、json、SQL、txt和xml格式的文件。

**6. 自定义应用风格**
根据个人偏好定制应用界面样式。
