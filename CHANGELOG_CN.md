# Changelog



## [0.0.2] - 2026-01-31

### 🚀 新特性
- **数据库管理**：支持自定义选择要在树形列表中显示的数据库。
- **编辑器体验**：新增 SQL 编辑器标签页关闭时的行为选项配置。

### ⚡ 优化
- **性能提升**：支持 SQL 文件的并行执行，显著提高大批量语句的处理速度。

### 🐞 Bug 修复
- 修复树节点无法搜索的问题
- 界面交互 ：
    - 修复了右侧面板内容无法实时刷新的问题。
    - 修复了导出数据和执行 SQL 文件时日志窗口不显示实时日志的 Bug。
    - 修正了数据导出进度中显示的表名不正确的问题。
- 数据库兼容性
  ：修复了 MySQL 解析器在处理包含反斜杠（\）的转义字符串时的解析错误。
